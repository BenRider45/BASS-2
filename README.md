# BASS-2

Welcome to BASS-2! Created by Benjamin Rider in collaboration with the Jin group lead by Dr. Dezhe Jin at Penn State University

## Installing and building

Eventually I intend to set up CI/CD pipelines through github so that all dependencies are included in a .exe/.dmg file. WIP

Currently, required dependencies are:
-qt6
-qwt 6.3

- all submodules (installed with command `git submodule update --init --recursive`)
- cmake
- mingw compiler (for windows)
